# Technology meets nature

> Однодневный пет-проект после курса по HTML и CSS, вдохновленный картинками на pinterest

---

## Демо

**[Посмотреть проект](https://webrvlad.github.io/technology-meets-nature)**  
**[Мой GitHub](https://github.com/webRvlad)**

---

## Особенности

- Верстка и анимация гексагонов
- Адаптив на vw и rem с плавным масштабированием независимо от разрешения экрана (хоть 8к)
- Сгенерированное нейросетью изображение и видео по ней
- Семантичная вёрстка и доступность
- В проекте один экран

---

## Технологии

- **HTML5** — семантическая разметка  
- **CSS3** — `clip-path`, вложенные селекторы  
- **JavaScript** — простое управление анимацией и видео  
- **Шрифты** — кастомный `TupoVyaz-Bold`  
- **Изображения** — в формате `avif`
